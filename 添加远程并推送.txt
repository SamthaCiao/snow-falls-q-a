# 添加 GitHub 远程仓库并推送

当前仓库没有配置远程，需要先添加再推送。

## 步骤 1：在 GitHub 上创建仓库（若还没有）

- 打开 https://github.com/new
- 填写仓库名（如 RAG_1.31），选 Public
- 不要勾选 "Add a README file"
- 点 Create repository，复制生成的仓库地址（如 https://github.com/你的用户名/仓库名.git）

## 步骤 2：在项目目录执行（请替换为你的仓库地址）

在 CMD 或 PowerShell 中进入项目目录后执行：

  cd /d "D:\work\和鲸社区\superlinear课程\AI architect\phaseB\RAG_1.31版本 上线后"

  git remote add origin https://github.com/你的用户名/你的仓库名.git

  git branch -M main
  git push -u origin main

若你的默认分支是 master，则用：

  git push -u origin master

## 说明

- origin：远程名称，一般都用 origin
- -u：设置当前分支的上游为 origin/main，之后直接 git push 即可
- 首次推送可能要求登录 GitHub（浏览器弹窗或凭据管理器）
